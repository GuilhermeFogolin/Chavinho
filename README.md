# Vila Kombat – O Duelo da Vizinhança

Um jogo de combate casual onde os icônicos personagens da Vila do Chaves resolvem seus desentendimentos de forma lúdica.

## 🕹️ Como Jogar

O jogo pode ser jogado por 1 ou 2 jogadores localmente no mesmo teclado.

### Controles Jogador 1 (P1):
- **W**: Pular
- **A / D**: Mover para Esquerda / Direita
- **F**: Atacar (Sanduíche, Boneca ou Bola dependendo do personagem)

### Controles Jogador 2 (P2):
- **Seta para Cima**: Pular
- **Seta Esquerda / Direita**: Mover
- **L**: Atacar

## 🎭 Personagens

| Personagem | Arma Principal | Estilo |
| :--- | :--- | :--- |
| **Chaves** | Sanduíche de Presunto | Equilibrado. Possui regeneração lenta de vida passiva. |
| **Kiko** | Bola de Futebol Americano | Longo alcance. Seus ataques atingem oponentes mais distantes. |
| **Chiquinha** | Boneca de Pano | Ágil. Movimentos mais rápidos e saltos mais altos. |

## 🛠️ Tecnologias Utilizadas

- CSS3 (Aestética Premium)
